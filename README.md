Để tạo site riêng cá nhân thầy cô dùng cho học sinh của mình ôn thi, quý thầy cô làm theo các bước sau:
Bước 1. Đăng kí một hosting và domain
Bước 2. Tải hai file index.php và admin.php lên thư mục gốc của hosting.
Bước 3. Truy cập vào đường link: http://tendomaincuaban/admin.php để tải lên và cấu hình đề thi (khuyên dùng là file PDF)
Bước 4. Gửi đường link http://tendomaincuaban cho học sinh để vào ôn tập.
